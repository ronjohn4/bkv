Application Configuration

Application Configuration allows application administrators direct access to the key/value pairs that control how applications operate.

Instance is the table that stores the Key Value pairs that are read ad used by applications. KeyVals are defined for each App/Env combination.

Key is short for Environments. Our current environments are PRD, UAT1, UAT2, UAT3 and STG.

Bags Each Key/Instance is grouped in a single bag.
